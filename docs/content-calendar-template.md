# 콘텐츠 캘린더 템플릿

## 운영 개요

### 채널별 발행 빈도
- **커뮤니티**: 매일 1-2개 (운영진 or 회원)
- **블로그**: 주 3회 (월/수/금)
- **인스타그램 피드**: 주 3-4회
- **인스타그램 릴스**: 주 2-3회
- **인스타그램 스토리**: 매일 2-3개
- **유튜브** (선택): 주 1회

### 콘텐츠 제작 워크플로우
```
1. 커뮤니티 인기글 모니터링
2. 주간 테마 선정
3. 블로그 포스트 작성 (SEO 최적화)
4. 인스타그램 카드뉴스 제작
5. 릴스/쇼츠 편집
6. 스토리 스케줄링
7. 크로스 포스팅
```

### 세부 콘텐츠 기획

#### 월요일: 브랜드 비교

**커뮤니티 게시글**
```
제목: [비교] 시몬스 vs 에이스 전격 비교 | 어떤 게 나을까?

내용:
1. 가격 비교 (모델별)
2. 기술 비교 (스프링, 소재)
3. 장단점
4. 추천 대상
5. 사용자 평가 (커뮤니티 투표 결과)

태그: #시몬스 #에이스침대 #매트리스비교
```

**블로그 포스트**
```
제목: 시몬스 vs 에이스 vs 삼분의일 비교 총정리 | 침대 커뮤니티

구성:
- 브랜드 소개
- 가격대별 라인업
- 3사 비교표
- 추천 모델 TOP 3 (각 브랜드)
- 실사용자 후기
- CTA: 더 많은 후기는 커뮤니티에서

키워드: 시몬스 에이스 비교, 매트리스 브랜드 비교
이미지: 5장 이상
```

**인스타그램 카드뉴스 (7장)**
```
1장: 커버 - "시몬스 vs 에이스 vs 삼분의일"
2장: 가격 비교
3장: 시몬스 특징
4장: 에이스 특징
5장: 삼분의일 특징
6장: 추천 대상
7장: CTA - 커뮤니티 링크

해시태그: #시몬스 #에이스침대 #삼분의일 #매트리스비교 #침대추천 (총 25개)
```

**인스타그램 스토리**
```
- 투표: "어떤 브랜드 쓰세요?"
  (시몬스/에이스/삼분의일/기타)
- 링크: 블로그 비교글
- 예고: 내일 삼분의일 개봉 영상 공개
```

#### 화요일: 실사용 후기

**릴스 (15초)**
```

자막: "압축 매트리스 개봉 순간 ㄷㄷ"

```

**스토리**
```

- 링크: 삼분의일 상세 후기 (커뮤니티)
```

#### 수요일: 정보성 콘텐츠

**블로그 포스트**
```

구성:
1. 포켓스프링 (특징, 장단점, 추천 제품)
2. 본넬스프링 (특징, 장단점, 추천 제품)
3. 라텍스 (특징, 장단점, 추천 제품)
4. 메모리폼 (특징, 장단점, 추천 제품)

6. 선택 가이드
```
